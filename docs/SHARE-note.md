**Claude 사용량 미터기 만들었어요 🖥️**

$25짜리 LILYGO AMOLED 보드(ESP32-S3)에 **Claude Code 세션/주간 한도를 실시간으로** 띄우는 데스크 미터기예요. 코딩하다 한도에 막히기 전에 흘깃 보고 속도 조절하려고 만들었습니다.

- **구조:** Mac이 사용량을 모아서(ccusage = 토큰/비용, statusLine 훅 = 진짜 한도%) → ESP32가 WiFi(또는 USB-C)로 받아 AMOLED에 표시
- **화면:** 세션 %(대형) + USAGE↔TIME 페이스 비교 + 주간 % + 번레이트(점 깜빡임)
- **삽질 포인트 3개:**
  1. `Plus`는 QSPI가 아니라 **SPI 변종** → 일반 예제로는 화면 깜깜
  2. 색이 **분홍으로 깨짐 = 엔디안 버그**(바이트 스왑으로 해결)
  3. 진짜 한도는 **statusLine 훅**으로 가져옴(공식 지원, 토큰 해킹 불필요)
